# Ultra Finder

This is a git/firebase/env Finder bot


## Install 

Install dependency:

```bash
  npm install fs
  npm install https
  npm install cli-progress
```


## Usage
```bash
> node [progName] [filename.txt]

in you txt file each line must be only [exemple : bybit.com
                                                  binance.com
                                                  laboutique.com
                                                  ]
no url preffix needed like "https://"                                                  
```
